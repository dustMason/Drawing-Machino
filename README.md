G-Code Commands Implemented
-----

- M114: calls `where`
- M18: disables motors
- T, B, R, L, G, H, I
- J: sets motor direction (as in config?)
- G00, G01, G0, G1:
  line drawing commands?
- X, Y, Z
- F: set feed rate (drawing speed?)
- G02, G2, G03, G3: `arc`
- G04, G4: `dwell` (pause?)
- D00: move one motor
- D01: adjust spool diameters (config)
- D02: return spool diameters
- D03, D04: SD card commands
